How to read this

  • Direction locked so far: all-caps wordmark, solid colors (SEMI white / DOPED orange), gradient reserved for the wafer mark, white logo overlays on video, wafer mark as the bug. Open question: which font.
  • Each font is shown at wordmark and full-logo scale, then on real frames so you can see it small.
  • For the persistent watermark, the white wafer mark or white full logo at ~72% opacity (a corner) reads cleanest over busy footage.
  • Lower-thirds use the display font for the name + IBM Plex Mono for the role line.

Fonts load live from Google Fonts (needs internet). Frame images (sd_frame.jpg, sd_frame_austin.jpg) sit beside this file. Once a font + treatment is chosen, clean SVG/PNG exports at broadcast sizes are the next step.
